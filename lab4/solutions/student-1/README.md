## Описание

Программа `calc` реализует:
- обработку аргументов командной строки;
- указатели на функции;
- вариативные функции (`va_list`);
- использование собственной библиотеки `libft`.

## Поддерживаемые операции

- `sum`
- `mul`
- `avg`

## Сборка

```bash
make
```

## Примеры запуска

```bash
./calc sum 1 2 3
./calc mul 2 3 4
./calc avg 10 20 30
```

## Проверка valgrind

```bash
valgrind --leak-check=full ./calc sum 1 2 3
```
