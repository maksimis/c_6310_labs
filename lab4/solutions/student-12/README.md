# Лабораторная работа №4 — Поиск по шаблону (Вариант 4)
## Описание:   
Программа ищет в наборе строк (аргументы или `stdin`) строки, удовлетворяющие заданному шаблону поиска.

## Формат вызова: 
```
./search <режим> <шаблон> [строки...]
```
## Режимы:
- `substr` — строки, содержащие подстроку `шаблон`
- `prefix` — строки, начинающиеся с `шаблон`
- `suffix` — строки, заканчивающиеся на `шаблон`
- `len` — строки, длина которых равна числу, заданному в `шаблоне` (шаблон преобразуется в число)

Если строки не переданы сразу в вызове, они читаются из `stdin` построчно

## Реализовано:
1. Массив структур, связывающих режим и указатель на функцию-предикат
2. Предикат принимает строку и шаблон (для `len` — целое число) и возвращает 1 при совпадении, в противном случае 0
3. Используются только функции из собственной библиотеки libft
4. В main всё согласно требованиям
5. Ошибки при неизвестном режиме или недостаточном количестве аргументов обработаны
6. Указатели на функции: Выбор режима (`substr`, `prefix`, `suffix`, `len`) через таблицу переходов.
7. Вариативная функция: `ft_error_log` принимает произвольное количество строк для вывода ошибок.
8. Обработка `argc/argv` и чтение из стандартного ввода.

## Проверка:
1. Проверка режима `substr` — строки, содержащие подстроку `шаблон`:
```bash
./search substr "ABC" "ABC" "xABCx" "BCA" "ABBCC"
```
Результат:
```
ABC
xABCx
```
2. Проверка режима `prefix` — строки, начинающиеся с `шаблон`:
```bash
./search prefix "ABC" "ABC" "xABC" "BC" "ABCx"
```
Результат:
```
ABC
ABCx
```
3. Проверка режима `suffix` — строки, заканчивающиеся на `шаблон`:
```bash
./search suffix "ABC" "ABC" "xABC" "BC" "ABCx"
```
Результат:
```
ABC
xABC
```
4. Проверка режима `len` — строки, длина которых равна числу, заданному в `шаблоне` :
```bash
./search len 3 "A" "AB" "ABC" "ABCD" "EFG"
```
Результат:
```
ABC
EFG
```
5. Режим, когда строки вводятся вручную в консоль
Запустить без списка строк:
```bash
$ ./search substr "abc"
```
Ввести слова:
abcdef
hello
abc
Программа будет выводить подходящие:
Результат:
```
abcdef
abc
```
Для завершения Ctrl+C
