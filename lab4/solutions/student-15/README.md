# stats — лабораторная работа C (вариант 3)

Программа вычисляет статистические показатели для набора целых чисел,
переданных через аргументы командной строки.

## Сборка

Требования:

- компилятор C (gcc)
- библиотека `libft` в виде подмодуля или в каталоге `./libft`

Сборка:

```bash
make
```

Будет собран исполняемый файл `./stats`.

Очистка объектных файлов:

```bash
make clean
```

Полная очистка (включая бинарник):

```bash
make fclean
```

Пересборка:

```bash
make re
```

## Формат запуска

```bash
./stats <метрика> [числа...]
```

Метрика — одна из:

- `min` — минимальное значение
- `max` — максимальное значение
- `sum` — сумма
- `avg` — среднее арифметическое (целое, округлённое вниз)

Числа — целые значения, записанные в виде строк.

Примеры:

```bash
./stats min 5 3 8 1 9
# вывод: 1

./stats max 5 3 8 1 9
# вывод: 9

./stats sum 5 3 8 1 9
# вывод: 26

./stats avg 5 3 8 1 9
# вывод: 5
```

## Обработка ошибок

Программа проверяет:

- недостаток аргументов (`argc < 3`)
- пустой список чисел
- неизвестную метрику
- некорректные числа (нецелые строки)
- слишком большое количество чисел (в данной реализации — больше 10)

При ошибках:

- сообщение выводится в `stderr` через функции `libft`
- программа завершает работу с ненулевым кодом возврата

Примеры:

```bash
./stats
# stderr: Error: not enough arguments

./stats foo 1 2
# stderr: Error: unknown metric

./stats min abc 1 2
# stderr: Error: invalid number
```
