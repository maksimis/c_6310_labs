# Лабораторная работа №4 — Статистика числового набора (Вариант 3)

## Описание:   
Программа вычисляет статистические показатели для набора целых чисел, переданных через аргументы командной строки.

## Формат вызова: 
./stats <метрика> [числа...]

## Метрики:
- `min` — минимальное значение
- `max` — максимальное значение
- `sum` — сумма всех чисел
- `avg` — среднее арифметическое (целое, округлённое вниз)

## Реализовано:
1. Таблица переходов — массив указателей на вариативные функции `g_metrics[]`, связывающий коды метрик с функциями-обработчиками.
2. Функции обработки массива чисел (`ft_min_arr`, `ft_max_arr`, `ft_sum_arr`, `ft_avg_arr`)
3. Используются только функции из собственной библиотеки `libft`
4. В `main`:
   - обработка `argc/argv`
   - проверка корректности аргументов
   -  проверка, что аргументы являются числами (`is_number()`)
   - преобразование строк в числа через `ft_atoi`
   - динамическое выделение памяти под массив чисел

5. Обработка ошибок:
   - недостаточное количество аргументов
   - неизвестная метрика
   - пустой список чисел
   - аргумент не является целым числом
   - ошибка выделения памяти

6. Указатели на функции:
   - выбор метрики реализован через таблицу переходов (`struct + function pointer`)
7. Вариативная функция:
   - реализована функция `compute`, демонстрирующая использование `va_list`
   - функции `min_impl`, `max_impl`, `sum_impl`, `avg_impl` демонстрируют использование `va_list`
8. Освобождение памяти (`free(numbers)`) и отсутствие утечек
