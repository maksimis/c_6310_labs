# Лабораторная работа №6 — Студент 8, Вариант 4 (анализ файла)

## Сборка

```bash
make
```

## Запуск

```bash
./analyze_file <filename>
```

## Пример

```bash
./analyze_file data/input.txt
```

Выводит 4 числа, каждое на новой строке:
1. Количество строк
2. Количество слов
3. Количество символов
4. Длина самой длинной строки

## Цели Makefile

- `make` / `make all` — сборка
- `make clean` — удаление объектных файлов
- `make fclean` — полная очистка
- `make re` — пересборка
