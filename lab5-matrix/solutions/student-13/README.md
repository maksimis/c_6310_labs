# Лабораторная работа 5 — библиотека для работы с матрицами
Студент 13
Вариант 1 — сложение и вычитание двух матриц.

## Задание

Реализована библиотека для работы с матрицами целых чисел.

Для варианта 1 реализованы операции:

- создание матрицы;
- удаление матрицы;
- вывод матрицы;
- сложение двух матриц;
- вычитание двух матриц.

## Структура матрицы

```c
typedef struct matrix_struct
{
	int	**matrix;
	int	rows;
	int	columns;
} matrix_t;
```

## Реализованные функции

```c
matrix_t	*create_matrix(int rows, int cols);
void		remove_matrix(matrix_t *A);
void		print_matrix(matrix_t *A);
matrix_t	*sum_matrix(matrix_t *A, matrix_t *B);
matrix_t	*sub_matrix(matrix_t *A, matrix_t *B);
```

## Сборка

```bash
make
```

После сборки создается исполняемый файл:

```bash
matrix_demo
```

## Запуск

```bash
./matrix_demo
```

## Проверка Makefile

```bash
make
make clean
make fclean
make re
```

Назначение целей:

- `make` — собирает проект;
- `make clean` — удаляет объектные файлы;
- `make fclean` — удаляет объектные файлы, исполняемый файл и `libft.a`;
- `make re` — полностью пересобирает проект.

## Обработка ошибок

Функции проверяют:

- некорректные размеры матрицы;
- `NULL` вместо матрицы;
- невозможность выделения памяти;
- несовпадение размеров матриц при сложении и вычитании.

При ошибке функции возвращают `NULL` или выводят сообщение об ошибке.

## Использованные возможности

В работе используются:

- динамическое выделение памяти;
- структура `matrix_t`;
- собственная библиотека `libft`;
- вывод через функции `libft`;
- сборка через `Makefile`.
