# Лабораторная работа 4 — функции с переменным числом аргументов
Студент 13
Вариант 1 — калькулятор с произвольным числом операндов.

## Задание

Программа принимает операцию и произвольное количество целых чисел.

Формат запуска:

```bash
./calc <операция> [числа...]
```

Поддерживаемые операции:

- `sum` — сумма всех чисел;
- `mul` — произведение всех чисел;
- `avg` — среднее арифметическое, целое число с округлением вниз.

## Сборка

```bash
make
```

После сборки создается исполняемый файл:

```bash
calc
```

## Запуск

```bash
./calc sum 1 2 3 4
./calc mul 2 3 4
./calc avg 2 3 4
```

## Примеры работы

```bash
./calc sum 1 2 3 4
# 10

./calc mul 2 3 4
# 24

./calc avg 2 3 4
# 3

./calc avg 1 2
# 1

./calc sum -5 10 15
# 20
```

## Обработка ошибок

Если указана неизвестная операция:

```bash
./calc div 1 2
# Error: unknown operation
```

Если среди аргументов есть некорректное число:

```bash
./calc sum 1 abc
# Error: invalid number
```

Если программа запущена без нужных аргументов:

```bash
./calc
# Error: usage: ./calc <sum|mul|avg> [numbers...]
```

## Проверка Makefile

```bash
make
make clean
make fclean
make re
```

Назначение целей:

- `make` — собирает проект;
- `make clean` — удаляет объектные файлы;
- `make fclean` — удаляет объектные файлы и исполняемый файл;
- `make re` — полностью пересобирает проект.
