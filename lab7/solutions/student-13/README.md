# Лабораторная работа 7 — многопоточность в C
Студент 13

## Задание
Реализована задача «обедающие философы» с использованием потоков и мьютексов.

В программе используются:

- 5 философов;
- 5 потоков `pthread`;
- 5 мьютексов для вилок;
- обычный `enum` для состояния философа;
- флаговый `enum` для вилок в руках;
- защита от взаимной блокировки.

## Суть задачи

Есть 5 философов, которые по очереди думают, берут вилки, едят и кладут вилки обратно.

Каждому философу нужны две вилки: левая и правая. Вилки являются общими ресурсами, поэтому доступ к ним защищается мьютексами.

## Состояния философа

```c
typedef enum e_state
{
	THINKING,
	HUNGRY,
	EATING
} t_state;
```

Состояния:

- `THINKING` — философ думает;
- `HUNGRY` — философ хочет есть;
- `EATING` — философ ест.

## Флаги вилок

```c
typedef enum e_philo_flags
{
	PHILO_NONE = 0,
	PHILO_HAS_LEFT = 1 << 0,
	PHILO_HAS_RIGHT = 1 << 1
} t_philo_flags;
```

Флаги нужны, чтобы хранить информацию о том, какие вилки сейчас находятся у философа.

## Защита от deadlock

Чтобы избежать взаимной блокировки, философы берут вилки в разном порядке:

- философы с четным номером сначала берут левую вилку, потом правую;
- философы с нечетным номером сначала берут правую вилку, потом левую.

Из-за этого не возникает ситуация, когда все философы одновременно взяли одну вилку и бесконечно ждут вторую.

## Компиляция

```bash
cc -Wall -Wextra -Werror -pthread philosophers.c -o philosophers
```

## Запуск

```bash
./philosophers
```

## Пример вывода

```text
Philosopher 1 is THINKING

Philosopher 1 is HUNGRY

Philosopher 1 took left fork
Philosopher 1 is EATING

```

Порядок строк может отличаться, потому что потоки выполняются параллельно.
