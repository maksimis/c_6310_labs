# Лабораторная работа №7: Обедающие философы

## Описание

Решение классической задачи синхронизации «Обедающие философы» с использованием потоков (pthread) и мьютексов.

## Особенности

- 5 философов (потоков)
- 5 вилок (мьютексов)
- Обычный enum для состояния философа (THINKING, HUNGRY, EATING)
- Флаговый enum для отслеживания вилок в руках (PHILO_NONE, PHILO_HAS_LEFT, PHILO_HAS_RIGHT)
- Защита от deadlock: чётные философы берут левую→правую, нечётные — правую→левую
- Безопасный вывод через мьютекс
- Корректное завершение после заданного числа приёмов пищи
