## Предупреждения -Wall

1. Неиспользуемая переменная
Строка: 52
Код: int x = 1;
Текст предупреждения: warning: unused variable 'x' [-Wunused-variable]
Объяснение: Переменная объявлена и инициализирована, но нигде не используется.

2. Неиспользуемый параметр функции
Строка: 56
Код: void warn2(int parametr) {
Текст предупреждения: warning: unused parameter 'parametr' [-Wunused-parameter]
Объяснение: Параметр функции объявлен, но внутри функции нигде не используется.

3. Неиспользуемая статическая функция
Строка: 61
Код: static void warn3(void) {
Текст предупреждения: warning: 'warn3' defined but not used [-Wunused-function]
Объяснение: Статическая функция определена, но нигде в файле не вызывается.

4. Отсутствие return в функции
Строка: 68
Код: }
Текст предупреждения: warning: control reaches end of non-void function [-Wreturn-type]
Объяснение: Функция должна вернуть значение, но выполнение доходит до конца без return.

5. Сравнение знакового и беззнакового
Строка: 74
Код: if (x < y) {
Текст предупреждения: warning: comparison of integer expressions of different signedness: 'unsigned int' and 'int' [-Wsign-compare]
Объяснение: В условии сравниваются целые числа разных типов: знаковый и беззнаковый.

6. Использование неинициализированной переменной
Строка: 82
Код: int v = x;
Текст предупреждения: warning: 'x' is used uninitialized [-Wuninitialized]
Объяснение: Переменная используется до того, как ей было присвоено начальное значение.

7. Деление на ноль
Строка: 88
Код: int x = 1 / 0;
Текст предупреждения: warning: division by zero [-Wdiv-by-zero]
Объяснение: В выражении выполняется деление на ноль, что является недопустимой операцией.

8. Вводящий в заблуждение отступ
Строка: 95
Код: if (x < 5)
Текст предупреждения: warning: this 'if' clause does not guard... [-Wmisleading-indentation]
Объяснение: Отступ создаёт впечатление, что строка относится к if, хотя на самом деле это не так.

9. Пустое тело if/else/while
Строка: 104
Код: if (x == 0);
Текст предупреждения: warning: suggest braces around empty body in an 'if' statement [-Wempty-body]
Объяснение: После условия if стоит точка с запятой, поэтому тело условия оказывается пустым.

10. Использование char как индекса массива
Строка: 111
Код: int d = mas[x];
Текст предупреждения: warning: array subscript has type 'char' [-Wchar-subscripts]
Объяснение: В качестве индекса массива используется переменная типа char.

## Предупреждения -Wextra (бонус)

1. Сравнение, которое всегда истинно/ложно
Строка: 121
Код: if (x < 0) {
Текст предупреждения: warning: comparison of unsigned expression in '< 0' is always false [-Wtype-limits]
Объяснение: Условие всегда ложно, потому что беззнаковое число не может быть меньше нуля.

2. Параметр присваивается, но не используется
Строка: 127
Код: void warn12(int max) {
Текст предупреждения: warning: parameter 'max' set but not used [-Wunused-but-set-parameter]
Объяснение: Параметру функции присваивается новое значение, но дальше оно нигде не используется.

3. Переменная присваивается, но не используется
Строка: 133
Код: int x;
Текст предупреждения: warning: variable 'x' set but not used [-Wunused-but-set-variable]
Объяснение: Переменной присваивается значение, но затем она нигде не используется.

4. Переполнение при константных вычислениях
Строка: 139
Код: int x = 2147483647 + 1;
Текст предупреждения: warning: integer overflow in expression of type 'int' results in '-2147483648' [-Woverflow]
Объяснение: Константное выражение вычисляется с выходом за допустимые пределы типа int.

5. Сдвиг на количество бит, превышающее размер типа
Строка: 145
Код: int x = 1 << 40;
Текст предупреждения: warning: left shift count >= width of type [-Wshift-count-overflow]
Объяснение: Число сдвигается на количество бит, которое не помещается в размер типа int.
