## Предупреждения -Wall

### 1. Неиспользуемая переменная (y)
- **Строка:** 18
- **Код:** `int y = x * 2;`
- **Текст предупреждения:** `warning: unused variable 'y' [-Wunused-variable]`
- **Объяснение:** Переменная `y` объявлена и инициализирована в функции `missing_return`, но нигде не используется. Это указывает на лишние вычисления в коде.

### 2. Отсутствие return в функции
- **Строка:** 19
- **Код:** `}`
- **Текст предупреждения:** `warning: non-void function does not return a value [-Wreturn-type]`
- **Объяснение:** Функция `missing_return` объявлена как возвращающая `int`, но не содержит оператора `return`. Функция достигает конца без возврата значения, что приводит к неопределенному поведению.

### 3. Пустое тело if
- **Строка:** 33
- **Код:** `if (scanf("%lf", &radius) != 1);`
- **Текст предупреждения:** `warning: if statement has empty body [-Wempty-body]`
- **Объяснение:** После условия `if` сразу стоит точка с запятой, что создает пустое тело. Результат `scanf` никак не обрабатывается, хотя должен проверяться на корректность ввода.

### 4. Отсутствие скобок в логическом выражении
- **Строка:** 49
- **Код:** `if (radius > 0 && radius < 1000 || radius == 0)`
- **Текст предупреждения:** `warning: '&&' within '||' [-Wlogical-op-parentheses]`
- **Объяснение:** Из-за приоритета операторов результат может быть неочевидным. Оператор `&&` имеет более высокий приоритет, чем `||`, поэтому выражение эквивалентно `(radius > 0 && radius < 1000) || radius == 0`. Компилятор рекомендует добавить скобки для ясности.

### 5. Использование char как индекса массива
- **Строка:** 59
- **Код:** `arr[ch] = 5.0;`
- **Текст предупреждения:** `warning: array subscript is of type 'char' [-Wchar-subscripts]`
- **Объяснение:** Использование `char` в качестве индекса массива может привести к проблемам, так как `char` может быть знаковым (от -128 до 127) в зависимости от платформы. Значение 'A' (65) может интерпретироваться по-разному.

### 6. Сравнение знакового и беззнакового типа
- **Строка:** 43
- **Код:** `if (signed_val < unsigned_val)`
- **Текст предупреждения:** `warning: comparison of integers of different signs [-Wsign-compare]`
- **Объяснение:** Сравниваются знаковый `int` (signed_val) и беззнаковый `unsigned int` (unsigned_val). Беззнаковые числа всегда неотрицательные, поэтому при сравнении с отрицательными числами результат может быть неожиданным.

### 7. Деление на ноль
- **Строка:** 54
- **Код:** `int division = 100 / 0;`
- **Текст предупреждения:** `warning: division by zero is undefined [-Wdivision-by-zero]`
- **Объяснение:** Попытка деления на ноль приводит к неопределенному поведению. Это грубая математическая ошибка, которая может вызвать аварийное завершение программы.

### 8. Неиспользуемая переменная (division)
- **Строка:** 54
- **Код:** `int division = 100 / 0;`
- **Текст предупреждения:** `warning: unused variable 'division' [-Wunused-variable]`
- **Объяснение:** Переменная `division` объявлена и инициализирована, но нигде не используется. Это еще одна неиспользуемая переменная в программе.

### 9. Использование неинициализированной переменной
- **Строка:** 62-66
- **Код:**
  ```c
  double uninit;
  if (radius > 1000) {
      uninit = 10.0;
  }
  printf("%f\n", uninit);
 - **Текст предупреждения:** `warning: variable 'uninit' is used uninitialized whenever 'if' condition is false [-Wsometimes-uninitialized]`
- **Объяснение:** Переменная `uninit` инициализируется только если radius > 1000. Если условие ложно (радиус ≤ 1000), переменная остается неинициализированной, но используется в `printf`, что приводит к неопределенному поведению.

### 10. Неиспользуемая функция
- **Строка:** 13
- **Код:** `static void unused_helper(void) { }`
- **Текст предупреждения:** `warning: unused function 'unused_helper' [-Wunused-function]`
- **Объяснение:** Функция объявлена как `static` (видима только в этом файле), но нигде не вызывается. Это мертвый код, который можно удалить.

### 11. Вводящий в заблуждение отступ
- **Строка:** 69-71
- **Код:** 
```c
  if (radius > 0)
    printf("Радиус положительный\n");
    area = PI * radius * radius;
```
- **Текст предупреждения:** `warning: misleading indentation [-Wmisleading-indentation]`
- **Объяснение:** По отступу кажется, что `area = ...` относится к `if`, но на самом деле это отдельный оператор, который выполняется всегда. Такое форматирование может ввести в заблуждение читающего код.


## Предупреждения -Wextra

### 1. Переменная присваивается, но не используется
- **Строка:** 27 и 77
- **Код:** `int result; и result = 42;`
- **Текст предупреждения:** `warning: variable 'result' set but not used [-Wunused-but-set-variable]`
- **Объяснение:** Переменной `result` присваивается значение 42 в конце программы, но это значение нигде не используется. Это указывает на лишние вычисления.

### 2. Неиспользуемый параметр функции
- **Строка:** `83-85`
- **Код:** `void unused_param_func(double r) { printf("Функция вызвана\n"); }`
- **Текст предупреждения:** `warning: unused parameter 'r' [-Wunused-parameter`
- **Объяснение:** Параметр `r` передан в функцию, но не используется в её теле. Это может указывать на ошибку в логике или лишний параметр, который не нужен для работы функции.
