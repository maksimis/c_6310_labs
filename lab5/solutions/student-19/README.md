# Лабораторная работа №5 — Вариант 3. Умножение двух матриц

## Описание:
Умножение матриц — операция линейной алгебры, при которой из матрицы A размера `m × k` и матрицы B размера `k × n` получается матрица C размера `m × n`.

Умножение возможно только при условии:

```text
A.columns == B.rows
```

Элементы результирующей матрицы вычисляются по формуле:

```text
C(i,j) = A(i,1) × B(1,j) + A(i,2) × B(2,j) + ... + A(i,k) × B(k,j)
```

Функция:

```c
matrix_t *mult_matrix(matrix_t *A, matrix_t *B);
```

возвращает:
- новую матрицу с результатом умножения;
- `NULL`, если матрицы несовместимы или произошла ошибка выделения памяти.
