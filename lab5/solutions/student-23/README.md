# Лабораторная работа №5 — Вариант 3. Умножение матриц

## Описание

Данная лабораторная работа реализует библиотеку `matrix` для работы с матрицами целых чисел.

Основная операция — **умножение двух матриц**.

Умножение возможно только если:

```text
A.columns == B.rows

Элементы результирующей матрицы вычисляются по формуле:

C(i,j) = Σ A(i,k) × B(k,j)

Функция:

matrix_t *mult_matrix(matrix_t *A, matrix_t *B);

Возвращает:

новую матрицу с результатом умножения;
NULL, если матрицы несовместимы или произошла ошибка выделения памяти.
