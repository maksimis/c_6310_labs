# Лабораторная работа №5 — Библиотека для работы с матрицами

**Студент:** 20  
**Вариант:** 4 — Сравнение матриц и анализ свойств

## Описание

Библиотека `matrix` реализует операции над матрицами:

- `create_matrix` — выделение памяти под матрицу
- `remove_matrix` — освобождение памяти
- `print_matrix` — вывод матрицы в stdout
- `eq_matrix` — сравнение двух матриц
- `is_symmetric` — проверка симметричности
- `is_diagonal` — проверка диагональности
- `is_identity` — проверка единичности

## Сборка и запуск

```bash
make
./matrix
```
