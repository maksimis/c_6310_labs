# Лабораторная работа №5 — Сравнение матриц и анализ свойств (Вариант 4)
## Описание:   
Сравнение матриц, проверка симметричной, диагональной и единичной матрицы

Сравнение матриц — две матрицы A и B равны, если они имеют одинаковые размеры и все соответствующие элементы равны:
A == B  ⇔  A.rows == B.rows  &&  A.cols == B.cols  &&  A(i,j) == B(i,j)  ∀ i,j

Симметричная матрица — квадратная матрица, равная своей транспонированной:
A = Aᵀ  ⇔  A(i,j) == A(j,i)  ∀ i,j

Диагональная матрица — квадратная матрица, у которой все элементы вне главной диагонали равны нулю:
A(i,j) == 0  при  i != j

Единичная матрица — диагональная матрица с единицами на главной диагонали:
A(i,j) == 1  при  i == j
A(i,j) == 0  при  i != j

Каждая функция возвращает `1`, если условие выполняется, и `0` в противном случае.
